## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/lilyelha2000-source/Bash_Project.git
```

### Navigate to the project

```bash
cd Bash_Project
```

### Make the script executable

```bash
chmod +x main.sh
```

### Run the project

```bash
./main.sh
```
